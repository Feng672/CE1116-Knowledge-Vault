## 📚 Idea/Concepto 

Es el recurso finito que actúa como la memoria RAM del modelo, determinando cuántos tokens puede procesar simultáneamente para predecir el siguiente. Entre mayor sea la ventana de contexto, un modelo de IA puede procesar entradas más largas e incorporar una mayor cantidad de información en cada salida, sin embargo, también se puede diluir la atención, provocando que el modelo tome atajos cognitivos o sufra alucinaciones al procesar demasiada información irrelevante. Asimismo, para agrandar la ventana de contexto, se tendría que duplicar cuatro veces más la potencia de procesamiento ya que como el modelo calcula relaciones entre cada token y todos los anteriores, tiene un costo computacional cuadrático.
